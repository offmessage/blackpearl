Flotilla Hardware InputsÂ¶

blackpearl provides lightweight wrappers around the hardware inputs that come with the Flotilla kits. It’s not possible to change the inputs through Python code, but these modules provide some sanitisation of the input, as well as some convenience methods for calculating useful things from the raw data (like RGB values from the colour sensor and compass headings from the motion sensor).
